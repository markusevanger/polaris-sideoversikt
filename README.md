# 📰 Sideoversikt for Polaris Media Sør 📊
Sjekk ut demo: https://sidesjekk.markusevanger.no

## 💡 Bakgrunn
Sommerjobben min er en bedrift som er ansvarlig for layout i flere norske lokalaviser, bl.a Agderposten og Varden. Koordinering av sideproduksjon er tidligere gjort gjennom ett delt excel ark og DMs mellom mediegrafikere. Denne webappens oppgave er å gjøre arbeidsoppgavene mer oversiktlige, og lage et grensesnitt ment for disse oppgavene. Tjenester som Trello er ikke ønsket av bedriften da de koster penger, og krever enda mer datasystemer, og brukertilganger. Bedriften har allerede enormt mange legacy systemer å forholde seg til. 

Nettsiden er optimalisert for både mobil, tablet og desktop.

## 🛠️ Verktøy og Teknologier brukt
### Frontend
- React
- Tailwind
- Typescript
- [Shadcn](https://ui.shadcn.com/)
- Recharts

### Backend
- NodeJS
- ExpressJS
- MongoDB
- Javascript
